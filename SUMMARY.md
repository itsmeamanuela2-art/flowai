## 📞 Support & Hilfe

**Falls du stuck bist:**
1. Schau in die entsprechende Dokumentation
2. Schau Heroku Logs: `heroku logs --tail`
3. Schau Stripe Dashboard (Webhook Logs)
4. Schau PostgreSQL Logs: `heroku pg:diagnose`

**Häufige Fehler:**
- `DATABASE_URL is not set` → PostgreSQL Add-on aktivieren
- `Stripe error` → Secret Keys überprüfen
- `Cannot find module` → Dependencies installieren
- `CORS error` → CLIENT_URL in Env Vars überprüfen
